### Fixed

- **Fixed the same macOS mangled-name identity mismatch in the
  `--sources`/`--build-info` source-ABI path** — `source_extractors/clang.py`'s
  `SourceEntity.identity()` still recorded clang's raw, Mach-O-ABI-decorated
  mangled name, while `clang_source_edges.py` builds the same TU's graph
  edges by reusing `call_graph.py`'s/`type_graph.py`'s identity functions
  (now normalized). A public inline function collected via `--sources`
  would have hit the identical dropped-`public_api_internal_dependency_added`
  bug the header-only `--header-graph` path had before that fix.

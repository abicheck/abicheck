- **`consumer_required_symbol_removed` `ChangeKind`** (ADR-044 P2 item 1) —

  suppressible finding for a consumer binary's own required dynamic symbol
  the new library no longer exports, instead of only a bespoke string
  special-cased by each reporter format.

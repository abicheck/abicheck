# Support

## Getting help

- **Bugs & feature requests** → [GitHub Issues](https://github.com/abicheck/abicheck/issues)
- **Questions & discussions** → [GitHub Discussions](https://github.com/abicheck/abicheck/discussions)
- **Security vulnerabilities** → [Private Vulnerability Reporting](https://github.com/abicheck/abicheck/security/advisories/new) (do **not** open a public issue)

## Reporting issues

Please use the GitHub issue templates and include:
- `abicheck --version` output
- Operating system and Python version
- Minimal reproduction steps
